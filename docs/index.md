## Overview

Community CMS is a modern Next.js-based Content Management System designed for community organizations. It features:

- 🏠 **Home Page** - Vibrant landing page with hero section and events
- 👥 **About Us** - Mission, vision, values, and team showcase
- 📅 **Events** - Upcoming and past events with registration
- 📧 **Contact** - Contact form and membership information
- 🌐 **Bilingual Support** - English and Kannada content
- 📱 **Responsive Design** - Works on all devices

## Tech Stack

- **Framework**: Next.js 16.2.7 (App Router)
- **Language**: TypeScript
- **Styling**: SCSS with BEM methodology
- **Icons**: React Icons
- **Fonts**: Geist Sans & Geist Mono
